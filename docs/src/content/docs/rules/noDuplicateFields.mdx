---
slug: rules/noDuplicateFields
title: noDuplicateFields
description: Disallow duplicate fields in selection sets.
---

| Property         | Value               |
| ---------------- | ------------------- |
| Config name      | `noDuplicateFields` |
| Default severity | `warn`              |
| Context          | Document            |
| In recommended   | Yes                 |

## What it checks

Disallow duplicate fields in selection sets.

## Examples

```graphql
# ❌ Bad — duplicate field
query GetUser {
  user {
    id
    name
    id
  }
}
```

```graphql
# ✅ Good — no duplicates
query GetUser {
  user {
    id
    name
  }
}
```

## Configuration

```yaml
extensions:
  graphql-analyzer:
    lint:
      rules:
        noDuplicateFields: warn
```
