---
title: strictIdInTypes
description: Require ID field in object types.
---

| Property         | Value             |
| ---------------- | ----------------- |
| Config name      | `strictIdInTypes` |
| Default severity | `—`               |
| Context          | Schema            |
| In recommended   | No                |

## What it checks

Require ID field in object types.

## Examples

```graphql
# ❌ Bad — object type without ID field
type User {
  name: String
  email: String
}
```

```graphql
# ✅ Good — object type with ID field
type User {
  id: ID!
  name: String
  email: String
}
```

## Configuration

```yaml
extensions:
  graphql-analyzer:
    lint:
      rules:
        strictIdInTypes: warn
```
