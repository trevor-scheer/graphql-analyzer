## [0.1.0-alpha.0] - 2026-01-20

Initial alpha release. This project is under active development and APIs may change.

### Added

- GraphQL Language Server with diagnostics, hover, goto definition, and find references
- CLI tool (`graphql`) for validation and linting
- MCP server for AI-assisted GraphQL development
- Configuration via `.graphqlrc.yaml`
- Support for embedded GraphQL in TypeScript/JavaScript
- Remote schema introspection
- Extensible linting engine with configurable rules
